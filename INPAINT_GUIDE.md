# 局部重绘功能使用指南

## 快速开始

### 1. 基本使用

```bash
nai -M 修复手部 [发送图片]
```

### 2. 收到调暗的图片后

- 保存这张暗图
- 使用任意画图工具(Windows画图、PS、SAI等)
- 用**白色画笔**(#FFFFFF)涂抹需要重绘的区域
- 保存涂好的图片

### 3. 发送涂好的图片

直接发送给机器人,等待生成结果

## 涂鸦技巧

### ✅ 正确做法

- 使用纯白色 (#FFFFFF, RGB 255,255,255)
- 涂抹范围略大于目标区域
- 可以随意涂抹,不需要太精确
- 可以涂满一大片区域

### ❌ 错误做法

- 使用灰色、淡色(会被忽略)
- 涂得太小,边缘可能不完整
- 使用渐变色、半透明色
- 涂抹后再压缩图片(可能损失白色信息)

## 参数调节指南

### Strength (-N)

| 值 | 效果 | 适用场景 |
|---|------|---------|
| 0.3-0.5 | 轻微调整 | 微调细节、修复小瑕疵 |
| 0.6-0.7 | 中等修改 | 修复手部、调整姿势 |
| 0.8-0.9 | 大幅改变 | 改变发色、更换背景 |

### Noise (-n)

| 值 | 效果 | 适用场景 |
|---|------|---------|
| 0.0-0.1 | 保留原图细节 | 只做轻微修复 |
| 0.1-0.2 | 平衡 | 大多数场景 |
| 0.2-0.3 | 增加变化 | 希望风格有所改变 |

## 故障排查

### 问题: 重绘后没有变化

**可能原因**:
1. 没有使用白色画笔
2. 画笔透明度不是100%
3. strength参数太低

**解决方法**:
```bash
# 检查画笔设置,确保RGB都是255
# 增加strength
nai -M 提示词 -N 0.9 [图片]
```

### 问题: 边缘有接缝

**可能原因**:
1. 涂抹范围太小,没覆盖到边缘
2. 图片被压缩,白色区域受损

**解决方法**:
- 涂抹时扩大范围
- 使用PNG格式保存涂鸦图
- 避免微信等应用压缩图片

### 问题: 等待超时

**可能原因**:
1. 涂鸦时间超过60秒
2. 忘记发送图片

**解决方法**:
- 重新发起命令
- 提前准备好涂鸦工具

## 高级技巧

### 1. 多区域重绘

同时修改多个区域,只需要把所有需要重绘的地方都涂白即可

```bash
nai -M 修复手和脸 -N 0.7 [图片]
# 在图上同时涂白手部和脸部
```

### 2. 渐进式修改

small先用低strength试试效果,不满意再提高

```bash
# 第一次尝试
nai -M 修改发型 -N 0.6 [图片]

# 如果不够明显,第二次
nai -M 修改发型 -N 0.8 [图片]
```

### 3. 结合不同模型

不同模型擅长不同风格

```bash
# 写实风格
nai -M realistic face [图片]

# 动漫风格  
nai4-5 -M anime style [图片]

# Furry风格
nai -m furry -M改成兽耳 [图片]
```

## 注意事项

1. **图片格式**: 建议使用PNG,避免JPEG压缩损失
2. **尺寸**: 自动对齐到64倍数,不需要手动调整
3. **超时**: 60秒内完成涂鸦并发送
4. **隐私**: 图片会上传到NovelAI服务器
5. **消耗**: 局部重绘消耗Anlas与普通生图相同
