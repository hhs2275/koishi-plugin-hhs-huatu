# 更新日志

## 最新版本

### v0.2.0

**新增功能：**

- ✨ **角色提示词功能（Characters）**
  - 支持 NovelAI v4/v4.5 模型
  - 使用 `-K` 参数指定多角色
  - 支持中英文分号分隔
  - 5×5 网格位置系统
  - 支持角色独立的负向提示词

- ✨ **会员系统增强**
  - 自动清理过期会员数据
  - 到期提醒推送功能
  - 会员调试工具
  - 支持查询其他用户状态
  - 不活跃用户自动清理

- ✨ **新增快捷指令**
  - `nai4c` - NovelAI v4 Curated 模型

- ✨ **调试日志开关**
  - 可配置是否输出调试日志
  - 默认关闭，需要时启用
  - 支持 Characters 和图片审核日志

**改进：**

- 🔧 优化命令参数解析逻辑
- 🔧 改进日志输出格式
- 🔧 增强错误处理
- 📝 完善文档和教程

**修复：**

- 🐛 修复命令参数被提示词消费的问题
- 🐛 修复各种边缘情况的错误
